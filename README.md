# EasyRun
패턴 인식을 통한 편리한 앱 실행(CoreML을 통한 launcher 어플리케이션)

# 프로젝트 소개
패턴을 그려서 앱을 실행할수 있게 하는 것을 목표로 한다.

1. PC를 통해 **패턴을 미리 학습**시킨다.
2. 사용자가 정해진 **패턴과 어플을 매칭**시킨다.
3. 정해진 패턴을 통해 **손 쉽게 어플을 실행**시킨다.
